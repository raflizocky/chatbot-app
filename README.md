## Prerequisites

-   Node.js >= 18.18

## Resources Used

- Model: [Google AI Studio](https://aistudio.google.com/prompts/new_chat)

## Installation

1. Clone the project.

2. Write your own `GEMINI_API_KEY` at `.env`. (You can use other AI Models)
  
3. Open terminal, then run:

   ```
   npm i
   ```

   ```
   npm run start
   ```

4. Open `localhost:3000` at browser.
